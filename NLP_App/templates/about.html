About Page

This is an app that I created to learn more about nlp and especially more about deployment. Here I deployed 2 NLP APIs. Sam is a simple conversational chatbot, created with pytorch. and John is question answering bot, based on pretrained HuggingFace model called distilbert, which is smaller version of Bert model, I converted this model into onnx model and quantized it for better interference. John takes 2 inputs: question and context. Model get question from user and context using wikipedia api.
